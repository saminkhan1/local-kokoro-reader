Local Kokoro TTS Reader for Mac
================================

Use this after installing Local Kokoro TTS Reader from the Chrome Web Store.

1. Double-click Install.command.
2. Wait for the installer to say it is done.
3. Open Chrome.
4. Open any webpage.
5. Click Local Kokoro TTS Reader.
6. Press Play.

The first install downloads the local voice engine and voice files. After that,
reading pages aloud runs on this Mac.

No account, subscription, paid API, or cloud text-to-speech service is used.

To remove the local voice service later, double-click Uninstall.command.
