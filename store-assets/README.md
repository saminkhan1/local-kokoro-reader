# Chrome Web Store Assets

Use these files in the Chrome Web Store developer dashboard.

- `store-icon-128.png` - store icon, 128x128
- `screenshot-1-read-page.png` - screenshot, 1280x800
- `screenshot-2-selected-text.png` - screenshot, 1280x800
- `screenshot-3-local-private.png` - screenshot, 1280x800

Run `./scripts/check.sh` before publishing. It verifies these dimensions.
